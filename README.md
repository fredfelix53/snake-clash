# 🐍 Snake Clash!

A modern, colorful snake game with power-ups, daily challenges, and combo scoring.

## Features
- **Smooth touch controls** — swipe to move, keyboard support for desktop
- **Three food types** — Regular (🍎), Gold (⭐ bonus points), Poison (💀 shrink)
- **Power-ups** — Speed Boost (⚡), Shield (🛡️), Magnet (🧲)
- **Level progression** — reach target score to advance, speed increases each level
- **Combo multiplier** — chain consecutive eats for bonus points
- **Coin system** — earn coins, spend on power-ups before each round
- **Daily challenge** — seeded food positions, same for everyone
- **Classic mode** — endless play
- **Rewarded ads** — double coins by watching ads (AdMob integrated on Android)

## Play
Open `game/index.html` in a browser or build the Android APK.

## Build APK
The GitHub Action builds the APK on every push. Download from Actions artifacts.
